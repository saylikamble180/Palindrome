# Palindrome Checker (Python)

This is a simple python program to check whether a number or string is a palindrome.

## Features
- Checks if input is palindrome
- Simple and easy logic

## Concepts Used
- Loops / string manipulation
- Conditional Statements

## How it works
The program reverses the input and compares it with the original value. If both are same, it is a palinndrome.

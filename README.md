# String Analyzer

A simple and interactive menu-driven Python application that allows users to perform multiple operations on a given string such as checking palindrome, counting vowels, reversing the string, and analyzing character frequency.

---

## ✨ Features

- Palindrome Checker – Checks whether the input string is a palindrome (case-insensitive)
- Vowel Counter – Counts the number of vowels in the string
- String Reversal – Reverses the input string
- Character Frequency Analyzer – Displays frequency of each character
- Dynamic Input Update – Allows entering a new string without restarting
- Menu-Driven Interface – Easy and interactive user experience

---

## 🛠️ Concepts Used

- Functions
- Loops
- Conditional Statements
- Dictionaries
- String Manipulation

---

## ⚙️ How It Works

1. User enters a string
2. A menu is displayed with different operations
3. User selects an option
4. The selected operation is performed on the string
5. The program continues until the user chooses to exit

---

## ▶️ How to Run

1. Clone this repository or download the file
2. Open terminal / command prompt
3. Run:

python main.py
